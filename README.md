# Research Tool RAG

## Overview
This project is a Retrieval-Augmented Generation (RAG) based research tool built in Python. It retrieves relevant information from stored documents and uses it to generate context-aware answers.

## Features
- Document-based question answering
- Retrieval-Augmented Generation pipeline
- Modular Python implementation
- Easy to extend for additional data sources

## Project Structure
RESEARCH-TOOL-RAG/
- resources/        Input documents or knowledge base
- main.py           Main entry point
- rag.py            Core RAG pipeline logic
- .env              Environment variables
- README.md         Project documentation

## Tech Stack
- Python
- RAG pipeline
- LLM integration
- Environment variables for API key handling

## How to Run
1. Install dependencies
2. Create a `.env` file with required API keys
3. Run:

```bash
python main.py
